### Linked lists

Arrays have some drawbacks. Linked lists solve some of these.

We'll build up a linked list implementation with these methods:
* `addFirst()`
* `addLast()`
* `removeFirst()`
* `size()`
* `first()`
* `last()`
* `isEmpty()`
